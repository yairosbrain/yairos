# Y.A.I.R.O.S. — מסמך ארכיטקטורה מלא

> **Your Artificial Intelligence Realtime Operating System**
> העוזר האישי בהשראת J.A.R.V.I.S — גלקסיית סוכנים תלת־ממדית, שליטה בקול בעברית ובאנגלית, נגיש מכל מכשיר, ורץ **100% על שכבות חינמיות**.

---

## 1. החזון במשפט אחד

אפליקציית PWA שנפתחת מכל מכשיר (טלפון, מחשב, טאבלט, טלוויזיה), מציגה "מוח" תלת־ממדי שבו כל צומת צבעוני הוא מחלקה־סוכן, מקשיבה לך בלחיצת כפתור, מתחקרת אותך ב־5 שאלות מדויקות על כל רעיון לאתר/אפליקציה — ואז בונה אותו: **פורסת אותו לבד לאינטרנט (מסלול א)** או **מגישה לך אפיון + פרומפט מושלם (מסלול ב)** — לבחירתך בכל פרויקט מחדש.


| שכבה | טכנולוגיה | תפקיד | עלות |
|------|-----------|-------|------|
| Frontend | **React + Vite + TypeScript** | האפליקציה עצמה | חינם |
| תלת־ממד | **Three.js** (react-three-fiber) | גלקסיית הסוכנים | חינם |
| PWA | **vite-plugin-pwa** | התקנה למסך הבית בכל מכשיר | חינם |
| מוח (ברירת מחדל) | **Puter.js** | גישה ל־Claude ועוד 400+ מודלים בלי מפתח | חינם (חשבון Puter אישי) |
| מוח (אופציה 2) | **Gemini API** | מפתח אישי מ־Google AI Studio | שכבה חינמית עם מכסה יומית |
| מוח (אופציה 3) | **Claude API** | מפתח אישי מ־Anthropic | ⚠️ בתשלום בלבד — אין שכבה חינמית |
| זיכרון | **Convex Starter** | פרויקטים, שיחות, תוצרים | חינם (1M קריאות/חודש, 0.5GB) |

| פריסת האתרים שנבנים | **GitHub Pages** | מסלול א — אתר חי בדקה | חינם (ריפו ציבורי) |

> **הערה חשובה על Claude API:** בניגוד ל־Puter (שנותן Claude בחינם) ול־Gemini (שכבה חינמית), חיבור ישיר ל־Claude API דורש טעינת קרדיט בתשלום אצל Anthropic. האופציה קיימת בהגדרות למי שירצה — אבל ברירת המחדל החינמית היא Puter.

## 5. שכבת המוח — Brain Layer (ההחלטה המרכזית)

עיקרון: **ממשק אחיד אחד** שכל המערכת מדברת איתו, ומתחתיו שלושה ספקים מתחלפים.

```ts
// src/brain/index.ts — הממשק האחיד
askBrain(messages, options) → Promise<string>

// src/brain/providers/
puter.ts    // ברירת מחדל — Claude דרך Puter, בלי מפתח
gemini.ts   // Gemini API ישיר עם מפתח אישי
claude.ts   // Claude API ישיר עם מפתח אישי (בתשלום)
```

**איך כל ספק עובד:**

| ספק | מה צריך | איך מתחברים |
|-----|---------|--------------|
| **Puter** (ברירת מחדל) | כלום. בפעם הראשונה קופץ חלון התחברות לחשבון Puter — נפתח עם yairosbrain@gmail.com פעם אחת | `puter.ai.chat()` ישירות מהדפדפן |
| **Gemini** | מפתח חינמי מ־Google AI Studio (עם אותו Gmail) | קריאת REST ישירה מהדפדפן עם המפתח |
| **Claude** | מפתח בתשלום מ־Anthropic Console | קריאה ישירה מהדפדפן עם כותרת `anthropic-dangerous-direct-browser-access` |

**כללי ברזל:**
- מסך ההגדרות כולל בורר ספק + שדות מפתח.
- מפתחות נשמרים **רק ב־localStorage של המכשיר** — לא בקוד, לא ב־GitHub, לא ב־Convex. מזינים פעם אחת בכל מכשיר.
- אם ספק נכשל (מכסה נגמרה / תקלה) — Yairos מודיע ומציע לעבור לספק אחר בלחיצה.

## 6. גלקסיית הסוכנים — גרסה 1: מפעל האתרים

כל סוכן = **פרומפט מערכת ייעודי + קריאה לשכבת המוח**. כל סוכן הוא כוכב בגלקסיה, בצבע המחלקה שלו. כשהוא רץ — הכוכב זוהר, והקו אל הסוכן הבא נדלק. בדיוק כמו בסרטון.

| סוכן | צבע בגלקסיה | תפקיד | קלט ← פלט |
|------|-------------|--------|------------|
| **YAIROS CORE** | 🔵 כחול (מרכז) | המנצח: מבין את הבקשה, מנתב בין המחלקות, מדבר איתך | הכול עובר דרכו |
| **INTERROGATOR** — תחקיר | 🟡 צהוב | מנסח 5 שאלות מדויקות ומותאמות לרעיון שלך | בקשה חופשית ← 5 שאלות |
| **ARCHITECT** — אפיון | 🟠 כתום | הופך את התשובות לאפיון: דפים, פיצ'רים, נתונים, טכנולוגיות | 5 תשובות ← מסמך אפיון |
| **DESIGNER** — עיצוב | 🟣 סגול | פלטת צבעים, טיפוגרפיה, פריסה, אווירה | אפיון ← שכבת עיצוב |
| **CODER** — קוד | 🟢 ירוק | מייצר את קבצי הקוד המלאים של האתר | אפיון+עיצוב ← קבצים |
| **QA** — בדיקות | 🔴 אדום | סורק את הקוד: שגיאות, רספונסיביות, RTL, נגישות | קבצים ← קבצים מתוקנים |
| **DEPLOYER** — פריסה | 🩵 טורקיז | מסלול א: ריפו + Pages אוטומטי. מסלול ב: חבילת אפיון+פרומפט | קבצים ← קישור חי / חבילה |

## 7. הזרימה המלאה — מ"היי Yairos" ועד אתר חי

1. לוחץ על כפתור המיקרופון ומדבר: *"Yairos, תבנה לי אתר תיק עבודות לצלם"*
2. זיהוי הדיבור ממיר לטקסט (he-IL) ← **CORE** מזהה כוונת "פרויקט חדש" ← נפתח פרויקט ב־Convex
3. **INTERROGATOR** מייצר 5 שאלות מותאמות בדיוק לרעיון ← Yairos שואל אותך בקול ובטקסט, אחת־אחת
4. אתה עונה (בקול או בהקלדה) ← כל תשובה נשמרת
5. **ARCHITECT** מסכם לאפיון ← **DESIGNER** מוסיף שכבת עיצוב ← האפיון מוצג לאישורך
6. Yairos שואל: **"מסלול א — אני בונה ופורס לבד לאינטרנט, או מסלול ב — אפיון ופרומפט מוכנים בשבילך?"**
7. **מסלול א:** CODER מייצר קבצים ← QA בודק ומתקן ← DEPLOYER יוצר ריפו, דוחף קוד, מפעיל Pages ← Yairos מקריא: *"האתר באוויר, אדוני"* + קישור חי
8. **מסלול ב:** נבנית חבילה — אפיון מלא + פרומפט בנייה מושלם ← כפתור העתקה/הורדה ← מביאים אליי (Claude) ובונים יחד
9. הפרויקט, השיחה והתוצרים נשמרים ב־Convex — נגישים מכל מכשיר, לנצח

## 8. מסלול א לעומק — פריסה אוטומטית בחינם

- בהגדרות מזינים פעם אחת **GitHub Fine-grained Token** (הרשאות מצומצמות: ריפוזיטוריז בלבד) — נשמר ב־localStorage

- תוך כדקה יש כתובת חיה: `https://yairosbrain.github.io/<שם-הפרויקט>`
- **עדכונים בהמשך:** אומרים *"Yairos, תשנה את צבע הכפתור לכחול"* ← אותו pipeline רץ ודוחף עדכון לאותו ריפו
- הערה: Pages חינמי דורש ריפו **ציבורי** — מתאים לאתרים שממילא עולים לאוויר

## 10. מודל הנתונים ב־Convex

```ts
// settings — העדפות גלובליות (בלי מפתחות! הם ב-localStorage)
settings: {
  language: "he" | "en",
  brainProvider: "puter" | "gemini" | "claude",
  voiceOutput: boolean,
}

// projects — כל אתר/אפליקציה שYairos בונה
projects: {
  name: string,
  request: string,              // הבקשה המקורית שלך
  status: "interrogating" | "spec" | "awaiting_choice"
        | "building" | "qa" | "deploying" | "live" | "delivered",
  questions: string[],          // 5 שאלות התחקיר
  answers: string[],
  spec: string,                 // מסמך האפיון המלא
  deployMode: "auto" | "manual" | null,   // מסלול א / ב
  repoUrl?: string,
  liveUrl?: string,
  createdAt: number,
}

// messages — כל השיחות (הטרנסקריפט)
messages: {
  projectId?: Id<"projects">,
  role: "user" | "yairos" | "agent",
  agentId?: string,
  text: string,
  lang: "he" | "en",
  ts: number,
}

// agentRuns — תיעוד כל ריצת סוכן (מה שמדליק את הגלקסיה)
agentRuns: {
  projectId: Id<"projects">,
  agent: "interrogator" | "architect" | "designer"
       | "coder" | "qa" | "deployer",
  input: string,
  output: string,
  status: "running" | "done" | "error",
  ts: number,
}
```

> בעתיד נוסיף טבלת `knowledge` — הארכיון האישי שלך (פתקים, רעיונות) — כשנרחיב את המוח מעבר לבניית אתרים.

## 11. ממשק המשתמש — ארבעה מסכים

**מסך Boot (כמו בסרטון):**
```
Y . A . I . R . O . S .

> YAIROS CORE ............... ONLINE
> NEURAL LATTICE ............ CALIBRATED
> KNOWLEDGE GALAXY .......... 7 NODES / 12 LINKS
> DOMAINS ................... WEB FACTORY
> SPEECH SYNTHESIS .......... READY
> AUDIO SENSORS ............. STANDING BY
> UPLINK — BRAIN ............ SECURE

          [ INITIALIZE ]
```

**מסך הגלקסיה (הלב):** כוכבים צבעוניים בתלת־ממד, קווים ביניהם, סיבוב/זום במגע ובעכבר. לחיצה על כוכב פותחת כרטיס מחלקה (כמו ARCHIVE RECORD בסרטון). למטה: שורת "שאל את Yairos…" + כפתור מיקרופון + כפתור ASK.

**מסך Transcript:** היסטוריית השיחה המלאה + צ'יפים מהירים ללחיצה (כמו COLD BREW CANS בסרטון).

**מסך Settings:** שפה, ספק מוח + מפתחות, טוקן GitHub, הפעלת קול.

## 12. PWA — התקנה בכל מכשיר
